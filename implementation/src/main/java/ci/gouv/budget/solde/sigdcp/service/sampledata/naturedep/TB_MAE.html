Le transit concerne les frais engendrés par le retour d'affectation (rappel des fonctionnaires du  Ministère des Affaires Etrangères) d'un fonctionnaire ou d'un agent de l'Etat.
Ces frais sont pris en charge par l'Etat de Côte d'Ivoire par l'intermédiaire de la Direction de la Solde.

Etapes à suivre pour une prise en charge de transit:

Etape 1: Constitution de dossier de diplomate

Le dossier comporte:

  •  la décision de rappel ;
  • le certificat de prise de service ; 
  • la facture pro forma; 
  • le format életronique des documents sus-cités.

Etape 2 : Ouverture de compte utilisateur
L'ouverture de compte se fait une seule fois sur l'espace SIGDCP.Si vous avez déjà un compte, alors passez à l'étape suivante.

  • Cliquez sur le bouton "Ouverture de compte";
  • Entrez les informations demandées;
  • Validez.

Etape 3 : Connexion

  • Cliquez sur le bouton"'Cliquer ici pour solliciter ce service";
  • Entrez votre nom utilsateur et votre mot de passe;
  • Validez.

Etape 4: Demande d'indemnisation

  • Remplissez le formulaire qui vous est proposé;
  • Enregistrez votre demande;
  • Cliquez sur le bouton "Soumettre".

Vous recevrez un message vous indiquant le résultat de votre demande.
Si le résultat est positif, vous devriez vous rendre dans les services de la SDDCP pour le dépôt physique de votre dossier.
